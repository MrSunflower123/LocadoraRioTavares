# LocadoraRioTavares
- Status do projeto: Concluído.

- Tecnologias aplicadas: Linguagem java, ApacheNetBeans 21, MySQL Workbench 8.0 .

- Desenvolvedor: MrSunflower123.

- Objetivo do software: Auxiliar no gerenciamento de jogos alugados pelos clientes de uma locadora.

- Funcionalidades do sistema:
* Cadastrar usuários que trabalharão com o sistema.
* Cadastrar o perfil de um cliente que deseja alugar um jogo.
* Cadastrar os dados de um jogo que está disponivel na loja fisíca.
* Cadastrar um empréstimo de jogo feito por um cliente.
* Monitorar o prazo de aluguel de um jogo e seu prazo de devolução.
* Apresentar todos os jogos que foram alugados por um cliente.
* Backup de todos os dados do sistema.
